# そば屋は心のヤバイやつ 〜ドキドキ好感度ADV〜

窓際族エリアの立ち飲み処を営む、白い仮面のそば屋さんを攻略する恋愛ノベルゲーム。
プレイヤーはアクシデンチュア株式会社の一般社員「あなた」。7つの選択で好感度ゲージを
満タンにすると、両想いエンド（クリア）が見られる。

## 遊び方
- 各シーンで **3つの選択肢** から1つを選ぶ（全7回）。
- 選択によって **好感度ゲージ（0〜100）** が変化する。どの選択がどれだけ効くかは、プレイして確かめよう。
- 7回の選択後、最終好感度でエンディングが分岐：
  - **100%（MAX）**: ★TRUE END★ 両想い（クリア）
  - **75〜99%**: HAPPY END 両想い一歩手前
  - **45〜74%**: GOOD END ビール仲間
  - **0〜44%**: NORMAL END 常連客

## 実装メモ（ネタバレ注意）
- 各選択肢には加点差があり、シーンごとの「ビールを渡す」系の選択が最も好感度を上げる（`src/scenes.ts` の `isBeer`）。ただし **UI・カバー・説明文ではこれを明示せず**、プレイヤーが自分で気づく設計にしている。

## シーン構成（全7話）
1. 赤坂の曲がり角（食パンダッシュで衝突）
2. 窓際族エリアの立ち飲み処で再会
3. メニューに伸びた手が触れる
4. 残業明けの差し入れ
5. 急な雨とひとつの暖簾
6. ウクレレライブの夜（夏祭り・花火）
7. 夜の屋台で告白

## 技術
- Vite + TypeScript（フレームワーク非依存のバニラ実装）。
- そば屋は設定のNG変更（白い仮面・ビールジョッキ）を守ったインラインSVGで描画。
- 世界観の禁止事項（ブラック企業・いじめ・パワハラ・鬱・グロ）は扱わない、コミカルな純愛ラブコメ。

## 開発
```bash
npm install
npm run dev      # 開発サーバ
npm run build    # tsc 型チェック + 本番ビルド（dist/）
npm run preview  # ビルド結果のプレビュー
```

## デプロイ
`GAMES_PORTAL/games.json` に `soba-romance` として登録済み。
`main` への push で `.github/workflows/deploy-games-pages.yml` が
`dist/` を `games/soba-romance/` に配置して GitHub Pages に公開する。
