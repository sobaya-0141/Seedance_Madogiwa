## Seedance Prompt Rules
- Seedance向けの動画生成プロンプトは、**セリフ（キャラクターの発話内容）以外はすべて英語**で記述する。
  - 理由: 日本語の説明文はSeedanceでの再現精度が落ちるため。
  - セリフのみ、日本語のまま `"..."` で引用して埋め込む（例: `shouting "島流し一択やろ！"`）。
- 台本ファイル内の「画面内容」「カメラ」等の解説は日本語のままでよい（人間が読むためのドキュメントのため）。あくまで実際にSeedanceへ渡す prompt コードブロックのみ英語化の対象。
