# Selected square-head concepts

These imagegen concepts define the Minecraft-like modeling direction.

- `takosan-square-head-a-detailed.png`: active Takosan shape reference
- `yametaro-selected-reference.png`: authoritative Yametaro reference supplied
  by the user; do not substitute the more pixelated face variant

Shared geometry constraints: perfect cube head, unbeveled rectangular body
parts, block-layer hair or hood, and rigid animation pivots.

Yametaro identity locks: asymmetrical black side-part hair, thick round black
glasses, completely empty pure-white lenses with no pupils, circular pink
cheeks, lavender shirt, light collar, black button, and blue botanical marks.
